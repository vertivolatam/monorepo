# Welcome to the docs

## How to add Formatting to your internal docs

According to the user guide of the MyST extension for Sphinx, the [admonitions](https://mystmd.org/guide/admonitions) are the named Markdown [directives](https://mystmd.org/guide/directives#directive-admonition) that can be used with this extension.

If you're familiar with Confluence article formatting, this is similar to [panels](https://support.atlassian.com/confluence-cloud/docs/insert-elements-into-a-page/#Panel) (as you may know them from the latest Confluence), or the [info, tip, note, and warning macros](https://support.atlassian.com/confluence-cloud/docs/insert-the-info-tip-note-and-warning-macros/).

Let me show you how to use directives to highlight any sections of your new articles.

```{note}
This is a note example.
```

```{seealso}
This is a see-also example.
```

```{important}
This is an important example.
```

```{hint}
This is a hint example.
```

```{tip}
This is a tip example.
```

```{warning}
This is a warning example.
```

```{attention}
This is an attention example.
```

```{caution}
This is a caution example.
```

```{error}
This is an error example.
```

```{danger}
This is a danger example.
```

```{code-block}
This is a scripting example.
```
