## 10 Buyer Personas

### Personas Residenciales (B2C)

#### #1 — Maria, La Mama Urbana Consciente

- **Demografia**: 34 anos, San Jose Costa Rica, Marketing Manager, casada con 2 hijos, ingreso ~$4,000/mes, tech media
- **Necesidades criticas**: Alimentos sin pesticidas para sus hijos, funcione sin conocimiento tecnico, quepa en apartamento
- **Plan**: Basic ([ver pricing]) + Vertivo Home ([precio redactado]) + Caja Basica ([ver pricing])
- **SRD**: user% 22%, revenue% 12%

#### #2 — Carlos, El Techie Entusiasta

- **Demografia**: 28 anos, Medellin Colombia, developer en fintech, soltero, tech muy alta (Home Assistant, IoT hobbyist)
- **Necesidades criticas**: API abierta para integrar, datos exportables en tiempo real, sistema estable
- **Plan**: Pro ([ver pricing]) + Vertivo Home ([precio redactado]) + on-demand ([ver pricing])
- **SRD**: user% 15%, revenue% 10%

#### #3 — Rosa, La Abuela Jardinera

- **Demografia**: 63 anos, Heredia Costa Rica, maestra jubilada, vive sola, tech baja (WhatsApp y Facebook)
- **Necesidades criticas**: Sin esfuerzo fisico, interfaz accesible (letras grandes), robot maneje todo automaticamente
- **Plan**: Basic ([ver pricing], pagado por su hijo) + Vertivo Home ([precio redactado], regalo) + Caja Basica ([ver pricing])
- **SRD**: user% 8%, revenue% 4%

!!! warning "Rosa Test"
    Si Rosa (63 anos, solo usa WhatsApp) no puede completar un flujo, el UX necesita revision. Este es el criterio de accesibilidad de Vertivo.

#### #8 — Andres, El Influencer de Barrio

- **Demografia**: 31 anos, Envigado Colombia, arquitecto + content creator (15K seguidores), tech alta
- **Necesidades criticas**: Programa de referral con incentivos, producto fotogenico, herramientas de sharing
- **Plan**: Pro ([ver pricing]) + Vertivo Home ([precio redactado]) + marketplace ([ver pricing])
- **SRD**: user% 10%, revenue% 6%

### Personas Comerciales (B2B)

#### #4 — Diego, El Restaurante Farm-to-Table

- **Demografia**: 41 anos, Ciudad de Panama, dueno de restaurante de 40 asientos, ingreso ~$15,000/mes
- **Necesidades criticas**: Reducir costos de hierbas ($800/mes actual), calidad consistente, API para POS
- **Plan**: Commercial ([ver pricing]) + Vertivo Pro ([precio redactado]) + Caja Comercial ([ver pricing])
- **SRD**: user% 12%, revenue% 20%

#### #5 — Valentina, La Emprendedora de Bienestar

- **Demografia**: 36 anos, Bogota Colombia, duena de estudio de yoga + juice bar, tech media
- **Necesidades criticas**: Estetica del invernadero, ingredientes cosechados frente al cliente, reducir costos
- **Plan**: Commercial ([ver pricing]) + Vertivo Pro ([precio redactado]) + marketplace ([ver pricing])
- **SRD**: user% 8%, revenue% 12%

### Personas Industriales y Especializadas

#### #6 — Roberto, Director Ops Agroindustrial

- **Demografia**: 48 anos, Ciudad de Mexico, COO de cadena de 15 restaurantes, ingreso >$20,000/mes
- **Necesidades criticas**: Integracion ERP/SAP, compliance COFEPRIS, SLA dedicado, ROI para el board
- **Plan**: Industrial ([ver pricing] + SLA) + 4x Vertivo Scale ([precio redactado]) + bulk ([ver pricing])
- **SRD**: user% 4%, revenue% 22%

#### #7 — Lucia, Agronoma Consultora

- **Demografia**: 39 anos, Cartago Costa Rica, agronoma con 12 clientes, tech alta en software agronomico
- **Necesidades criticas**: Dashboard de fleet, Expert Mode (robot sugiere, ella decide), escalar sin contratar
- **Plan**: Commercial ([ver pricing]) + $0 hardware (clientes compran)
- **SRD**: user% 3%, revenue% 4% (multiplicador)

#### #9 — Directora de Comedor Escolar (B2G)

- **Demografia**: 52 anos, San Jose Costa Rica, 400 estudiantes, presupuesto MEP, tech baja
- **Necesidades criticas**: Suministro constante, precio accesible, trazabilidad para auditorias
- **Vertical**: Urban-Farming-as-a-Service o Product-as-a-Service
- **SRD**: user% 3%, revenue% 3%

#### #10 — AgriData Corp (Ecosystem)

- **Demografia**: Empresas AgTech, universidades, ministerios, laboratorios R&D
- **Necesidades criticas**: Datos geo-tagged unicos de rendimiento aeroponico en LATAM, API con SLA
- **Plan**: Custom enterprise ($2,500+/mo)
- **SRD**: user% 1%, revenue% 5%
